Hi! Welcome to DDLT, an upcoming mod for Doki Doki Literature Club!

Disclaimer: This mod is not a replacement for the original game. Please play through the original game first before playing this mod.

Instructions:
1) Download DDLC from ddlc.moe
2) Copy these mod files over to the DDLC game. (If it asks to replace files, click "yes".)
3) Start the game using DDLT.exe instead of DDLC.exe.

Author's Notes: This mod is one of my first mods, so I apologise if anything seems out of place.
